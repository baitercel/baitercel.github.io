三阶魔方试炼总结

发表于 2017-01-31 | 分类于 魔方 | | 阅读次数

三阶魔方我初玩魔方是想教我的小侄子，才自学的魔方然后教给他，后来发现自己深深喜欢上了魔方。教程 我是直接找的教程，如果自己琢磨，自己研究完成的话，估计要很久，而且也没有那个耐性，所以直接百度教程，魔方小站，就好了，（题外话，魔方小站的站长周博是清华大学物理系的硕士，很牛的一个人），里面有很多的教程。

初级玩法

最简单的初级玩法就是层先法，1层、2层、3层这样，我最开始学的就是这个，半天就学会了，不过拼完用时是很慢的，忘了几分钟了，反正挺长的。小站站长讲解的很通俗易懂，方法我不过多的说了，直接打开魔方小站站长的网站就有了。

高级玩法

高级玩法就是CFOP了，大概几百个公式，一下学起来有点慢，小站站长写出了一个简版的CFOP，20多个公式吧，初学玩家，想进一步提升速度的话，可以先学习简版的CFOP然后再过渡到CFOP，学完简版的CFOP，并且熟练应用，魔方完成时间进入1分钟以内很容易的，然后就是CFOP 完整版，去魔方小站下载CFOP公式，然后然后公式表一个一个的练就好了。

实战演练

我玩魔方断断续续的大概得有1年时间了，目前的水平也就是1分左右，1分钟的时间大部分浪费在了F2L上面了，我一般做F2L需要的时间大概是40秒，做完F2L然后到拼完大概用时20秒左右，接下来我会苦练F2L。我分享下我从一个14岁三阶13秒一个魔友那里交流来的教程，每天一个F2L ，两个OLL ，魔方打乱成公式的样子，然后再按照公式还原，一定要练的非常熟练。学习魔方是一个枯燥乏味的过程，学习久了肯定会厌烦的，厌烦的时候，可以换4阶玩玩，总之就是一个坚持的过程，一个肌肉记忆。

购买魔方

三阶魔方我有三个，两个普通的几块钱，一个贵的是75元，gan356Air，普通的魔方我就不说了，我说下gan356Air这个魔方，据说是比赛专用的魔方，买完魔方之后，千万别加油，我就是直接加油，导致油多特别黏，然后不得已买的酒精擦拭掉多余的油才好使的。买完之后都是厂家直接加油加好的。并不是油多就一定好。

总结：坚持！坚持！坚持！

风雨哈弗路-读后感

发表于 2017-01-29 | 分类于 读书 | | 阅读次数

题记：

这本书买很早了，一直没读，今年春节准备消化掉，用了两天就读完了。

讲述内容：

莉丝 ,未成年，父母吸毒，生活困顿，无法上学，四处流量浪，
然后她下决心好好学习，决定重返校园，她边打工边学习，最终拿到了<<纽约时报>>的奖学金，
考上了哈佛，走向成功，毕业后她成立公司，四处演讲，
帮助那些与她有着同样困难的人重回校园。

读后感：

莉丝的不屈不挠，不向命运低头，积极乐观的态度，看过这本书的人,

应该都会受到鼓舞,确实再想想我们现在的生活，我想大多人都要比莉丝好太多，
起码不愁吃穿。我想每一个人都希望成为人上人，受人敬仰、尊敬，那我们就需要太多的努力，
有的时候不光努力就一定成功，
这些年我一直提醒自己一件事，千万不要自己感动自己。大多人看似的努力，不过是愚蠢导致的。什么熬夜看书到天亮，连续几天只睡几小时，多久没放假了，
如果这些东西也值得夸耀，那么富士康流水线上任何一个人都比你努力多了。
人难免天生有自怜的情绪，唯有时刻保持清醒，才能看清真正的价值在哪里,
我现在已经戒烟戒酒2年了，每天健身，游泳等运动，只想保持一个清醒的自己，
自己的努力一定要获得价值。回首2016年，有太多的计划没有完成，看了一下自己所列的todolist，
还在那里没有完成，无非就是自己的懒惰在作怪，唯一坚持下来的就是运动和跳舞，
17年的我比16年确实强壮了，希望17年的我在原来运动的基础上，完成自己其他的计划，并且完成一些作品。
书中大篇幅描写了莉丝的穷苦生活，小篇幅描写了，莉丝在新高中的事情，
初看此书我更感兴趣的是莉丝是如何在辍学那么长时间的情况下考上哈弗，书中却一笔带过，
我也感兴趣的是， 卡洛斯以后的生活，书中也没有提及，我是欣赏卡洛斯的，他是那么的爱莉丝，
但是毒品是他变了，但文中的描写依然能看出，吸毒之后的卡洛斯是依然爱这莉丝,
总之，风雨哈弗路这本书，确实是给了我些触动，也使我在大年初二制定春节在家的学习计划，
和教侄子学习的计划，并且大年初1晚上用了一些时间，完善了博客，的一些功能，这个风雨哈弗路的读后感也是我博客的第一篇文章。 16再见,17,我要继续努力

文章内容摘要：

我不能把萨曼莎从她的家庭中拯救出来，但是我可以做她的朋友。
我不能改变卡洛斯，但是我们可以保留那份友谊，让自己好起来。
我不能治愈我的家人，尽管我很想，但是我可以原谅他们，爱他们。
我还可以选择努力为自己创造一种生活，这种生活绝对不会被我的过去所束缚。
看着邮递员越来越近，我意识到这封信，这封来自哈佛的信，无论里面写了什么，
都不会打破我现有的生活。而且，我渐渐明白，无论这儿会发生什么，将来会发生什么，
我的生活绝对不会被外部条件所控制。如以往一样，我的意愿会决定我的生活一步一步向前。
